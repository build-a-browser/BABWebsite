# BuildABrowser Website

Source code for the BuildABrowser website, for which a URL will be added to this README once it is hosted.

BuildABrowser's website is written using Astro, allowing for embedding markdown files and templating at compile-time,
but allowing the website to be hosted statically and run without required JavaScript support (which BuildABrowser does not
support at this time).

To run, `npm run dev`.

To build, `npm run build`.
